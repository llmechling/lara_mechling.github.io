Why is it useful to apply a log transformation?¶

Log functions will show a relative change in customer spending, where the base historgram showed to absolute change in customer spending. For example, the absolute graph shows the exact number of dollars the customer spending changed, but the log transformation shows the percentage of relative change in customer spending. In this example I think you would need to know exactly what information you are looking to glean from the histograms. If you want to know how much the majority of your customers are spending I would use the graph of the absolute values of spending. If you want to know the relative change in customer sending as it increases, use the logrithmic graph. (I do not know if this is correct.
